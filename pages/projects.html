Best of 2024

Best of 2024 was a project for school, that had a focus on html, css, javascript and using APIs. The main challenge was integrating the provided API to dynamically load and display data about the best of 2024. I designed the site to fetch information from the API and present it in an interactive and visually appealing way, ensuring a seamless user experience. This project helped me strengthen my skills in working with APIs, as well as in building responsive and accessible web interfaces.
