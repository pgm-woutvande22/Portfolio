Jammin

Jammin is a Kanban board application designed to streamline task management. The frontend is built with React and CSS, providing a clean and responsive user interface. On the backend, it leverages JavaScript along with the headless CMS Strapi for efficient data handling and API management. With Jammin, users can add, edit, delete, and move tasks, with all updates seamlessly synced to the backend. This makes it a flexible and intuitive tool for organizing workflows and boosting productivity.
